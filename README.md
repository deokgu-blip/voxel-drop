# Voxel Drop (physics prototype)

A physics/connectivity variant of a 3D voxel blaster puzzle. Octopus cannons shoot
voxels by color; every voxel must stay connected to the central **core** or it
breaks off and tumbles away. Cut the right "bridge" color and whole chunks collapse.

**Play:** open the GitHub Pages link (single self-contained HTML).
- Click a queued octopus to deploy it into a slot — it auto-fires its color.
- Drag to rotate the model and find hidden colors.

Prototype build — work in progress.
